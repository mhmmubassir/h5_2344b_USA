# Phenotypic flexibility, rather than optimization, governs pandemic potential

Repository for the analyses, figures, processed data products, and supplementary materials associated with the manuscript:

**Mubassir et al. (2026)**  
*Phenotypic flexibility, rather than optimization, governs pandemic potential*

## Overview

This repository contains the computational materials used to support a manuscript examining how phenotypic diversity emerges, persists, and is pruned during the recent North American expansion of highly pathogenic avian influenza A H5N1 clade 2.3.4.4b.

The study integrates sequence-based, structure-based, and function-based analyses across a large H5 hemagglutinin (HA) dataset to investigate how standing phenotypic breadth may contribute to cross-species transmission potential.
